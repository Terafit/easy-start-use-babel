1)Заходим на сайт babel, далее Setup и выбираем CLI
2)Установка

    1. Создаём проэкт - npm init
    2. npm install --save-dev @babel/core @babel/cli

3)Настраиваем package.json. Делаем вот так вот
 
 "scripts": {
    "build": "babel src -d build"
  },

src - папка, в которой находится проэкт указываем её название
build - это папка, куда мы переместим уже скампелированый(под другую версию синтаксиса) код

4)Настраиваем конфиг babel(Нужно для того, чтобы babel знал в какую версию компелировать(в нашем случае в ES5))

    Во первых устанавливаем все пресеты: (https://babeljs.io/docs/en/babel-preset-env)

        команда: npm install --save-dev @babel/preset-env
    
    Во вторых создаём файл " .babelrc " и закидываем туда вот такой вот пресет:

    {
        "presets": [
            "@babel/preset-env"
        ]
    }

    Этот присет позволяет нам скомпелировать код в стандарт ES5

5)Запуск компеляции:

    В файле package.json мы измеяли код в scripts

    "scripts": {
        (((("build")))))):---------
    },

    И указали, что build будет запускать компеляцию

    Поэтому в командной строке пишем: npm run build

    npm run имяСкрипта
